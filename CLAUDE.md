# CLAUDE.md — Инструкции для AI-ассистента

## Проект

Коммерческий юридический лендинг компании «Профессионалы» (Омск).
Услуги: ликвидация фирм, реорганизация, регистрация ООО, изменения в ЕГРЮЛ.

## Стек технологий

- **HTML** — единый `index.html` (вся разметка, стили и скрипты в одном файле)
- **Tailwind CSS** — подключён через CDN
- **Кастомный CSS** — встроенный `<style>`, ~1500 строк (градиенты, glass morphism, 3D-эффекты, анимации blob, параллакс)
- **Vanilla JavaScript** — встроенный `<script>`, ~1350 строк (без фреймворков)
- **Google Fonts** — Montserrat (400, 500, 600, 700, 800)
- **Formspree** — обработка форм
- **Яндекс.Карты** — iframe карта офиса

## Структура

## Секции лендинга (порядок в index.html)

1. Header / Навигация (лого, меню, телефон, бургер)
2. Hero (заголовок, подзаголовок, CTA, изображение с параллаксом)
3. Блок доверия (статистика: 10+ лет, 300+ проектов и т.д.)
4. Услуги (3 карточки: ликвидация, реорганизация, регистрация)
5. Процесс работы (4 шага)
6. Цены (прайс-лист + Яндекс.Карта)

8. Footer / Контакты

## Правила работы

### Язык общения
- Все объяснения, комментарии и ответы — **только на русском языке**
- Код (HTML, CSS, JS) — без перевода, как есть

### Редактирование
- **Не переписывать файл целиком** — править только нужные места
- Использовать точечные правки (Edit tool), а не полную перезапись

### Дизайн и визуал
- Стиль: **минимализм, премиум**, без перегруза
- **Сохранять все анимации и визуальные эффекты** (blob, параллакс, 3D-карточки, glass morphism, cursor spotlight)
- Тёмная тема: фон `#0a0e1a`, акценты — оттенки синего (`#1e40af`, `#3b82f6`, `#60a5fa`)
- Glass morphism: полупрозрачные фоны с backdrop-blur

### Адаптивность
- **Мобильная версия важнее десктопной** (mobile-first)
- Основной брейкпоинт: 768px
- Учитывать `prefers-reduced-motion` для доступности

### JavaScript
- Только **ванильный JS**, без фреймворков и библиотек
- Сохранять существующие обработчики и логику (параллакс, аккордеон, модалки, форматирование телефона, UTM-метки)

### Формы
- Обработка через Formspree
- Есть honeypot-защита от спама
- Валидация перед отправкой
